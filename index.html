// Steps I Followed

  • 1 Created Azure SQL Database — provisioned Basic tier (5 DTUs, 2GB) with Microsoft Entra authentication and public endpoint networking
  • 2 Created Azure Static Web App — Free tier, connected to GitHub repository for automatic CI/CD deployments
  • 3 Configured GitHub Actions — Azure auto-generated a workflow file that deploys every code push to the live URL
  • 4 Added Connection String — configured SQLAzure connection string in Static Web App settings to securely link the app to the database
  • 5 Deployed and verified — confirmed live URL, GitHub Actions workflow, and connection string configuration
